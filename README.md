# Student Mental Health Analysis

![Mental Health](mentalhealth.jpg)

A SQL analysis exploring whether length of stay affects mental health scores for international students at a Japanese university.

## About

This project uses SQL to analyze survey data from international students, looking at:
- Depression scores (PHQ-9)
- Social connectedness scores (SCS)
- Acculturative stress scores (ASISS)

## Key Finding

International students in their first year make up the largest group (95 students), with mental health scores that vary by length of stay.

## Data

| Column | Description |
|--------|-------------|
| `stay` | Length of stay in years |
| `todep` | Depression score |
| `tosc` | Social connectedness score |
| `toas` | Acculturative stress score |
